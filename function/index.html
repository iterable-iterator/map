    Applies a given callable to each of the elements of the input iterable.

    Params:

    NameTypeAttributeDescription

    The callable to use.
    iterable Iterable

    The input iterable.

    Return:

    IterableIterator

    Example:

    // return [ 0 , 1 , 4 , 9 ]

    import pick from '@iterable-iterator/map/src/pick.js'

    Yields some of the properties of the input object. The properties to yield are designated by the input iterable.

    Params:

    NameTypeAttributeDescription
    object Object

    The input object.
    iterable Iterable

    The input iterable.

    Return:

    IterableIterator

    Example:

    // returns [ 'a' , 'e' , 'c' ]
list( pick( 'abcde' , [ 0 , 4 , 2 ] ) ) ;
